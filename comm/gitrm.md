## git rm

![](/assets/rm.png)


**git rm** - используется в Git для удаления файлов из индекса и рабочей копии. Она похожа на *git add* с тем лишь исключением, что она удаляет, а не добавляет файлы для следующего коммита.


``````bash=
git rm 
``````
